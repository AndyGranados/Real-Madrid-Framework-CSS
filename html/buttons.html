Componente

Botones

El componente .btn es la base de todos los botones. Combínalo con modificadores de variante y tamaño.

En esta página

Variantes de color

Cuatro variantes que siguen la paleta del framework. Cada una tiene estados hover y active definidos.

👁️ Vista previa
HTML
<button class="rm-btn rm-btn-azul rm-btn-md">btn-primary</button>
<button class="rm-btn rm-btn-dorado rm-btn-md">btn-accent</button>
<button class="rm-btn rm-btn-blanco rm-btn-md">btn-light</button>
<button class="rm-btn rm-btn-borde rm-btn-md">btn-outline</button>
ClaseFondoTextoHover

.btn-accentDorado #FEBE10Azul MadridDorado más oscuro + glow
.btn-lightBlanco #FFFFFFAzul MadridGris muy claro

Tamaños

Tres tamaños disponibles. Siempre se combina el modificador de tamaño con el de variante.

👁️ Vista previa
HTML
<button class="rm-btn rm-btn-azul rm-btn-sm">btn-sm</button>
<button class="rm-btn rm-btn-azul rm-btn-md">btn-md</button>
<button class="rm-btn rm-btn-azul rm-btn-lg">btn-lg</button>
ClasePaddingFont-size
.rm-btn-sm0.25rem 0.5rem0.875rem (14px)
.rm-btn-md0.5rem 1rem1rem (16px)
.rm-btn-lg0.75rem 1.5rem1.25rem (20px)
💡
Buena práctica: Siempre usa los tres modificadores juntos: class="rm-btn rm-btn-[variante] rm-btn-[tamaño]". La clase .btn sola no aplica color ni tamaño.
Estados

Los botones reaccionan a hover y active automáticamente. El estado disabled se puede aplicar con el atributo HTML o la clase.

👁️ Vista previa — intenta hacer hover
HTML
<!-- Normal -->
<button class="rm-btn rm-btn-azul rm-btn-md">Normal</button>

<!-- Disabled con clase -->
<button class="rm-btn rm-btn-azul rm-btn-md rm-deshabilitado">Disabled</button>

<!-- Disabled con atributo HTML -->
<button class="rm-btn rm-btn-dorado rm-btn-md" disabled>Disabled</button>

Ficha de jugador

Selecciona una acción para continuar con el formulario de inscripción.

HTML
<div class="card">
  <div class="card__body">
    <div class="card__title">Ficha de jugador</div>
    <p class="card__text">Selecciona una acción...</p>
    <div class="rm-flex rm-gap-2">
      <button class="rm-btn rm-btn-azul rm-btn-md">Confirmar</button>
      <button class="rm-btn rm-btn-borde rm-btn-md">Cancelar</button>
    </div>
  </div>
</div>

Nombres propios del framework

Además de los nombres genéricos, Madrid CSS ofrece aliases con identidad propia. Son equivalentes exactos — úsalos para hacer tu código más expresivo.

👁️ Vista previa
HTML
<button class="rm-btn rm-btn-azul rm-btn-md">btn-madrid</button>   <!-- = rm-btn-azul -->
<button class="rm-btn rm-btn-dorado rm-btn-md">btn-dorado</button>   <!-- = rm-btn-dorado  -->
<button class="rm-btn rm-btn-blanco rm-btn-md">btn-blanco</button>   <!-- = rm-btn-blanco   -->
<button class="rm-btn rm-btn-borde rm-btn-md">btn-contorno</button> <!-- = rm-btn-borde -->
Alias MadridEquivalente genéricoColor
.btn-madrid.btn-primaryAzul Madrid #00529F
.btn-dorado.btn-accentDorado Madrid #FEBE10
.btn-blanco.btn-lightBlanco Madrid #FFFFFF
.btn-contorno.btn-outlineContorno azul transparente
