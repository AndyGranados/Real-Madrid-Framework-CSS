Utilidades

Helpers

Clases de utilidad para spacing, display, tipografía, sombras y más. Sin escribir CSS adicional.

Espaciado (Margin & Padding)

Escala del 1 al 5. Los prefijos son: m-, mt-, mb-, ml-, mr- para margin y p-, pt-, pb-, pl-, pr- para padding.

EscalaValor remValor px aprox
-10.25rem4px
-20.5rem8px
-31rem16px
-41.5rem24px
-53rem48px
👁️ Ejemplo — spacing visual
mb-1 (4px)
mb-2 (8px)
mb-3 (16px)
mb-4 (24px)
HTML
<div class="rm-mb-3">Margin bottom 1rem</div>
<div class="rm-mt-4 rm-mb-2">Margin top 1.5rem, bottom 0.5rem</div>
<div class="rm-p-3">Padding uniforme 1rem</div>
<div class="rm-pt-2 pb-4 pl-3">Padding individual</div>
