Real Madrid CSS Framework

Un framework CSS moderno inspirado en la identidad del Real Madrid C.F. — construido con SASS, basado en componentes reutilizables y pensado para desarrolladores que quieren velocidad con estilo.

¿En qué está inspirado?

Real Madrid CSS toma su identidad visual directamente del Real Madrid Club de Fútbol — uno de los clubes más laureados de la historia. Sus tres colores institucionales definen toda la paleta del framework.

Blanco Madrid
#FFFFFF · $color-primary
Azul Madrid
#00529F · $color-secondary
Dorado Madrid
#FEBE10 · $color-accent
Success
#28a745 · $color-success
Danger
#dc3545 · $color-danger
ℹ️
Los colores están definidos como variables SASS en _variables.scss. Puedes personalizarlos fácilmente para adaptar el framework a cualquier proyecto.
Instalación y uso

Incluye el CSS compilado en tu HTML y comienza a usar las clases del framework al instante.

📄 HTML — Estructura base recomendada
HTML 
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="scss/main.css">
</head>
<body>

  <nav class="navbar">
    <div class="navbar__container">
      <a href="#" class="navbar__brand">Mi Proyecto</a>
    </div>
  </nav>

  <div class="rm-contenedor rm-mt-4">
    <div class="rm-fila">
      <div class="rm-col-12 rm-col-md-6">
        <h1>Hola Madrid CSS!</h1>
        <a href="#" class="rm-btn rm-btn-azul rm-btn-md">Comenzar</a>
      </div>
    </div>
  </div>

</body>
</html>
Estructura SASS

El framework está organizado siguiendo la arquitectura 7-1 simplificada de SASS.

📁 Estructura de archivos
SASS 
scss/
├── main.scss              ← Punto de entrada principal
├── _variables.scss        ← Colores, tipografía, breakpoints
├── _mixins.scss           ← respond(), flex-center(), button-base()...
├── base/
│   └── _reset.scss        ← Reset CSS normalizado
├── layout/
│   └── _grid.scss         ← Sistema de 12 columnas responsivo
├── components/
│   ├── _buttons.scss      ← .rm-btn, .rm-btn-azul, .rm-btn-dorado...
│   ├── _cards.scss        ← .card, .card__body, .card__title...
│   ├── _navbar.scss       ← .navbar, .navbar__brand...
│   ├── _alerts.scss       ← .alert--success/danger/warning/info
│   ├── _badges.scss       ← .badge, .badge--pill
│   ├── _forms.scss        ← .form__control, .form__label...
│   └── _tables.scss       ← .table, .table--striped, --hover
└── utilities/
    ├── _helpers.scss      ← Espaciado, display, texto, stats, navcards...
    └── _footbal.scss      ← .rm-field, .rm-player ⚽

Mixins disponibles

Los mixins encapsulan lógica repetible. Úsalos en tus propios archivos SASS.

MixinParámetrosQué hace
respond($bp)md o lgMedia query responsiva según breakpoint
flex-center—display:flex + justify + align center
button-base—Estilos base comunes para botones
madrid-glow—Box-shadow dorado del Real Madrid
madrid-border—Borde 2px dorado
container—Contenedor centrado max-width 1200px
Características del framework
📐
Grid de 12 columnas
Sistema responsivo con breakpoints md (768px) y lg (1024px). Clases .rm-col-N, .rm-col-md-N, .rm-col-lg-N.
🧩
Componentes BEM
Botones, cards, navbar, alertas, badges, forms y tablas con nomenclatura BEM clara y predecible.
⚽
Modo Fútbol
Módulo especial _footbal.scss para renderizar una cancha y posicionar jugadores con CSS puro.
