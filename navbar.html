Navbar

Barra de navegación responsiva con hamburger CSS-only — sin JavaScript. Gradiente azul Madrid, brand dorado y animación X al abrir.

Estructura BEM
Clase / ElementoDescripción
.navbarBarra completa — gradiente azul, sombra
.navbar__containerContenedor flex interno (max 1200px)
.navbar__brandLogo/nombre en dorado, 1.6rem, bold
.navbar__toggle-checkinput[type=checkbox] oculto — controla el menú móvil
.navbar__toggle-labelÍcono hamburger visible solo en móvil. Contiene 3 <span>
.navbar__navLista de links — columna en móvil, fila en md+
.navbar__linkLink con subrayado dorado animado al hover
.navbar__link--activeModificador para marcar página actual

En pantallas menores a 768px aparece el ícono ☰ hamburger. Haz click para desplegar el menú — funciona sin JavaScript.
HTML
<nav class="navbar">

  <!-- Checkbox oculto: controla apertura del menú -->
  <input type="checkbox" class="navbar__toggle-check" id="nav-toggle">

  <div class="navbar__container">

    <!-- Brand -->
    <a href="index.html" class="navbar__brand">⚽ Madrid CSS</a>

    <!-- Botón hamburger (solo móvil) -->
    <label class="navbar__toggle-label" for="nav-toggle">
      <span></span>
      <span></span>
      <span></span>
    </label>

    <!-- Links -->
    <ul class="navbar__nav">
      <li><a href="index.html" class="navbar__link navbar__link--active">Inicio</a></li>
      <li><a href="buttons.html" class="navbar__link">Botones</a></li>
      <li><a href="grid.html" class="navbar__link">Grid</a></li>
    </ul>

  </div>
</nav>
¿Cómo funciona sin JavaScript?

El truco usa el selector CSS :checked combinado con el selector de hermano adyacente ~. Cuando el usuario hace click en el <label>, marca el input[type=checkbox] oculto. CSS detecta ese estado y cambia el display del menú.

💡
El selector clave:
.navbar__toggle-check:checked ~ .navbar__container .navbar__nav { display: flex; }

Cuando el checkbox tiene :checked, el menú pasa de display:none a display:flex. Las 3 líneas del hamburger se animan a una X con transform: rotate().
